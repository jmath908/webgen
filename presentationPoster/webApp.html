Web App Design

Data Source

Gene expression data from  The Cancer Genome Atlas (TCGA)

Data Retrieval

Data was retrieved from the Genomic Data Commons via the   Firebrowse API, developed by the Broad Institute.

Application Features

Thus far, our efforts have been focused on the design of the following features:

  • Dynamic calling of available cancer types from TCGA
  • Gene expression heatmap
  • Gene expression violin plot
